Note:- download Node.js first if not installed then follow below steps

Run Code :-
1) Create Any folder
2) open terminal
3) clone this repo
4) write "cd HRMS"
5) write "clear"
6) press "Enter" 
7) write "npm init -y"
8) press "Enter"
9) write "npm i mongoose express nodemon ejs dotenv"
10) press "Enter"
11) write "npx nodemon server.js"
12) press "Enter"
13) Boom! app started
